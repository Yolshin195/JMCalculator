# JMCalculator
Тестовое задание на обучение в Java Mentor

## Что бы скомпилировать проект
```
  javac src/*.java
```

## Для зборки проекта
```
  nvm package 
```

## Для запуска тестов
```
  nvm test
```

## Для запуска проекта
```
  nvm package 
  java -jar target/JMCalculator-1.0-SNAPSHOT.jar 1 + 7
```
